What is the KrakenD Designer

The KrakenD Designer is configuration builder, released as an open-source javascript application. It helps you create and edit krakend.json files to get started with the API Gateway in a visual way.

It is a pure static page that does not send any of your configurations elsewhere nor track its contents. It's hosted online for convenience, but you can also run it locally.

Use this page together with a KrakenD Watch image to apply the changes in your local server automatically.

Documentation

What can I do with KrakenD Designer?

The KrakenD Designer generates a single configuration file. It is meant to become familiar with the JSON file and contains many number of functionalities.

Because the interface might already look overwhelming given the number of features it displays, the most advanced flags and some features are not configurable here. Still, if you import and export files with these hidden attributes, the designer respects them.

The main limitations of this tool are:

  • You cannot work with Flexible Configuration (multiple configuration files)
  • Use non-JSON formats, like YAML or TOML
  • Administer a production gateway. This is meant to be a development tool.
  • It does not check the validity of what you type.

Development tools

The KrakenD designer outputs valid configurations respecting the schema. But we encourage you to edit the JSON file by hand and spend some time understanding its structure. There are a few resources that might help you:

  • Understanding the configuration file
  • Hot reloading the configuration
  • IDE integration
  • Validating the configuration with check
